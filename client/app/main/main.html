Network Performance Test

Run a Test!

To run a throughput performance test, do the following:

  1. Select a client interface
  2. Run the following command on a laptop connected via Ethernet to your WiFi access point to start an Iperf server: iperf3 -s
  3. Provide the IP address of the Iperf3 server
  4. Select a transport layer protocol
  5. Click the Start Test button
